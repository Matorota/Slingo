#  Slingo — Song Management App

Slingo is an Android **song management application** built in **Kotlin**, designed to let users manage songs, customize profiles, and interact with a clean, modern interface.  
This project is currently **work in progress** and evolving quickly.

---


##  Features

###  Current Features
- **User Accounts**
  - Add and update profile images
  - Change user information from a dedicated settings tab

- **Song Management**
  - Manage songs inside the app
  - Store and organize song data
  - Backend logic for CRUD-style song operations (in progress)

- **UI & Navigation**
  - Updated profile page
  - Discovery page with new design changes
  - Smooth navigation and layout enhancements

- **YouTube API Integration**
  - YouTube API configured in `settings.gradle.kts`
  - Prepares the app for fetching song/video data from YouTube

---

##  Tech Stack

- **Kotlin** (100%)
- **Android Studio**
- **Gradle Kotlin DSL (`build.gradle.kts`)**
- **Material Design UI**
- **YouTube API** (configured)
